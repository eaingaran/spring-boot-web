Welcome to Jemos Spring Boot Web sample app

The goal os this project is to provide a simple and easy template for a Spring MVC application using Spring Boot. What are the advantages of using Spring Boot?

  • Zero XML configuration for your Spring MVC App
  • Easy Maven setup in a jar project which runs your Web application
  • Full support for unit and integration testing of your webapp without the need to start and stop containers
