Welcome to Jemos Spring Boot Web sample app

The goal os this project is to provide a simple and easy template for a Spring MVC application using Spring Boot. What are the advantages of using Spring Boot?

  • Zero XML configuration for your Spring MVC App
  • Easy Maven setup in a jar project which runs your Web application
  • Full support for unit and integration testing of your webapp without the need to start and stop containers

For years developers have struggled to get an easy way to setup and test a Spring MVC application. Typically one had to:

  • Setup a quite convoluted pom.xml with all necessary Spring dependencies
  • Setup web.xml to register the Spring Dispatcher Servlet
  • Setup Spring application context files, controller setup files, mappings, annotations, etc
  • Come up with all sorts of boilerplate code to just be able to simulate a web request to a web controller with a proper request and response

Although eventually this would work, it would require a lot of effort, especially on the testing front. I'm sure you've been there, googling ways to test a controller in an integration test with a proper request and response.

Now, imagine that your Spring MVC is contained in a Maven jar project, that your controller is a simple Java class with few annotations and that you can exercise it through a fully flagged servlet container like Tomcat or Jetty without having to start anything before your test or shutdown anything after the test has completed.

This represents the holy-grail of web applications speed of development and testing consistency.
